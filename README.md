# CleanClose AI: Autonomous B2B Sales Architecture 🤖💼

An enterprise-grade autonomous sales agent built to qualify leads, conduct discovery workflows, and negotiate high-ticket commercial cleaning contracts without human intervention. 

This repository serves as an architectural showcase. The core proprietary negotiation logic, prompt engineering, and API routing are maintained in a secure, private environment to protect intellectual property.

## 🏗️ Technical Stack
* **LLM Engine:** Anthropic Claude (Optimized for complex B2B reasoning)
* **Data Validation:** Pydantic (Strict JSON schema enforcement)
* **Design Pattern:** Spec-Driven Development & State Machine routing
* **Environment:** Python 3.10+ 

## 📊 Telemetry & Performance
The agent is designed not just to chat, but to drive measurable Monthly Recurring Revenue (MRR). All interactions are tracked through a commercial telemetry dashboard.

![Telemetry Dashboard](assets/Telemetry%20Dashboard.png)

## 🎯 Use Case: Commercial Facilities Management
The agent handles the complete top-to-middle funnel workflow for a commercial cleaning agency:
1. **Qualification:** Eliminates leads under the $2,500/mo minimum budget.
2. **Discovery:** Identifies specific pain points (e.g., cross-contamination, SLA failures).
3. **Negotiation:** Defends pricing based on hospital-grade sanitization standards.
